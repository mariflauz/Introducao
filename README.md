# Introducao
### Sobre o projeto: 
Repositório criado com o objetivo de apresentar os conceitos básicos de Git e GitHub como tarefa da aula de mesmo tema da entidade PET do curso de Sistemas de Informação.
### O que é Git?
O Git é um sistema de controle de versão.
Ele permite acompanhar mudanças no código ao longo do tempo, facilitando o trabalho em equipe e a organização dos projetos.

### O que é GitHub?
O GitHub é uma plataforma online que utiliza o Git para armazenar e compartilhar projetos.
Ele também permite colaboração entre desenvolvedores.

### Conceitos importantes:
- Repository (repositório): local onde o projeto fica armazenado
- Commit: registro de uma mudança no projeto
- Branch: uma “ramificação” do projeto para desenvolver alterações sem afetar a versão principal
- Merge: junção das alterações de uma branch com a principal
